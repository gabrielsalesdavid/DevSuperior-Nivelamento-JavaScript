6. **Objetos** - Criação, acesso e manipulação
7. **Funções** - Declaração, expressão, arrow functions

11. **Tratamento de Erros** - Try/catch/finally e tipos de erro

11. **Tratamento Avançado de Erros** - Erros personalizados

## 📖 Como Usar

### 1. Clonar o Repositório

```bash
git clone https://github.com/gabrielsalesdavid/DevSuperior-Nivelamento-JavaScript.git
cd DevSuperior-Nivelamento-JavaScript
```

### 2. Explorar a Documentação

Comece lendo a documentação de fundamentos e conceitos:

```
Docs/
├── Fundamentos/JavaScript-Fundamentos.md (Leia primeiro!)
└── Conceitos/JavaScript-Conceitos.md (Depois este)
```

### 3. Estudar os Tópicos Básicos

Explore os arquivos em `Topicos Basicos/` em ordem:

```javascript
// Exemplo: Executar um arquivo com Node.js
node "Topicos Basicos/tipos-dados.js"
```

### 4. Praticar com Exercícios

Resolva os exercícios propostos:

```javascript
// Exercício de Tópicos Básicos
node "Topicos Basicos/Exercicios/terceirizados.js"

// Problemas
node "Topicos Basicos/Problemas/maior-numero.js"
```

### 5. Executar o Projeto ViaCEP

Abra o arquivo HTML no navegador:

```bash
# No navegador, abra:
Projeto DS Via CEP/index.html
```

Ou use um servidor local:

```bash
# Com Python
python -m http.server 8000

# Com Node.js (com http-server instalado)
npx http-server
```

### 6. Estudar JavaScript Assíncrono

Entenda promises e async/await:

```javascript
// Estude nesta ordem:
// 1. set-timeout.js - Entender callbacks
// 2. promises.js - Promises e then/catch
// 3. async-await.js - Sintaxe moderna
// 4. fetch-api.js - Requisições HTTP
```

### 7. Aprofundar em OOP

Estude orientação a objetos:

```javascript
// Estude nesta ordem:
// 1. Objetos e Modulos/prototype.js
// 2. Objetos e Modulos/classes.js
// 3. Objetos e Modulos/modulos.js
// 4. Objetos e Modulos/Exercicios/ - Práticas
```

## 📦 Dependências

Este projeto utiliza apenas **JavaScript vanilla** (sem frameworks ou bibliotecas externas), com exceção de:

- **Fetch API** - Para requisições HTTP (API nativa do navegador)
- **DOM APIs** - Para manipulação de elementos (API nativa do navegador)

**Compatibilidade:**
- ✅ Node.js 14+
- ✅ Navegadores modernos (Chrome, Firefox, Safari, Edge)
- ✅ ES6+ (ECMAScript 2015 e posteriores)
